# spore-documentation-theme

代码文档皮肤，用于 spore-kit

## Getting Started

```shell
npm i -g documentation
npm i -D spore-documentation-theme
documentation build ./index.js -c ./documentation.yml -f html -o ./doc -t ./node_modules/spore-documentation-theme/dist/index.js
```
